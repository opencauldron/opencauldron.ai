OpenCauldron is built for teams. Invite your team members, track who's generating what, and manage access.

## Inviting users

Users sign up through your configured authentication provider (GitHub, Google, etc.). Admins can manage user roles from **Settings > Users**.

## Roles

- **Admin** — Full access, can manage settings, users, and brands
- **Member** — Can generate images and view the shared gallery

## Shared gallery

All generations are visible in the shared gallery. Team members can browse, search, and download each other's work. Use the leaderboard to see who's generating the most.

## Usage tracking

Admins can view per-user API usage and costs from the **Usage** dashboard. Set a daily generation limit per user to stay within budget.
